ASSETS PENDIENTES DE SUBIR MANUALMENTE
=======================================

Los siguientes archivos deben ser creados y subidos manualmente a esta carpeta:

1. favicon-32.png
   - Tamaño: 32x32 px
   - Contenido: el mismo bot SVG (fondo #0A0E17, cuerpo #C9A961)
   - Referenciado en: <link rel="icon" sizes="32x32">

2. favicon-192.png
   - Tamaño: 192x192 px
   - Contenido: el mismo bot SVG
   - Referenciado en: <link rel="icon" sizes="192x192">

3. apple-touch-icon.png
   - Tamaño: 180x180 px
   - Contenido: el mismo bot SVG (con padding interior generoso)
   - Referenciado en: <link rel="apple-touch-icon">

4. og-cover.jpg
   - Tamaño: 1200x630 px
   - Contenido: imagen de portada para redes sociales (Open Graph / Twitter Card)
   - Formato: JPG, calidad 85+
   - Referenciado en: og:image y twitter:image

NOTA: El favicon.svg ya existe en esta carpeta y es funcional
en navegadores modernos sin necesidad de los PNG.

Paleta de colores de referencia:
  - Fondo oscuro:  #0A0E17
  - Dorado oliva:  #C9A961
  - Dorado claro:  #D9BB7A
  - Blanco suave:  #FAFAFA
